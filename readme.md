# 🎵 Music Player Interface

A modern Music Player Interface built using HTML, CSS, and JavaScript. The application provides a clean UI with playback controls, playlists, albums, artists, analytics, and user settings.

## 📌 Features

- 🔐 Login & Logout
- 🎵 Music Library
- ▶️ Play / Pause Songs
- ⏭️ Next & Previous Song
- ❤️ Favorite Songs
- 🔀 Shuffle
- 🔁 Repeat
- 🔊 Volume Control
- 📀 Albums
- 🎤 Artists
- 📂 Playlists
- 📈 Analytics Dashboard
- ⚙️ Settings
- 📱 Responsive Design

## 🛠️ Technologies Used

- HTML5
- CSS3
- JavaScript (ES6)
- Chart.js
- Font Awesome
- Google Fonts

## 🚀 How to Run

1. Download or clone the repository.
2. Open the project in Visual Studio Code.
3. Install the Live Server extension.
4. Right-click on `index.html`.
5. Select **Open with Live Server**.
